# Contributing

Thanks for your interest in contributing to Awesome Agent Skills!

## Adding a Skill to the Awesome List

1. Fork the repo
2. Add your skill to the appropriate category in `README.md`
3. Use the format: `- [name](url) — One-line description`
4. If no existing category fits, suggest a new one
5. Submit a PR

### Requirements

- Skill must have a public README
- Skill must be publicly accessible on GitHub (or similar)
- Skill must be functional and installable

## Contributing a Custom Skill

1. Fork the repo
2. Create a new directory under `skills/` with your skill name
3. Add a `SKILL.md` file following the [Claude Code skill format](https://github.com/anthropics/claude-code/blob/main/docs/skills.md)
4. Submit a PR with a description of what the skill does

### Skill Structure

```
skills/
└── your-skill-name/
    └── SKILL.md
```

## Questions?

Open an issue and we'll help you out.
